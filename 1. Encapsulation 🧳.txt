3. Inheritance ğŸ§¬

Definition:
Inheritance allows a class to acquire properties and methods from another class, making code reusable.

PHP Example:

<?php
class Animal {
    public $name;

    public function __construct($name) {
        $this->name = $name;
    }

    public function makeSound() {
        echo "$this->name makes a sound.\n";
    }
}

class Dog extends Animal {
    public function makeSound() {
        echo "$this->name barks.\n";
    }
}

$dog = new Dog("Buddy");
$dog->makeSound(); // Buddy barks.
?>


Dog inherits from Animal but can override its behavior.

Definition:
Polymorphism means â€œmany forms.â€ It allows objects of different classes to be treated through the same interface while behaving differently.

PHP Example:

<?php
abstract class Shape {
    abstract public function area();
}

class Circle extends Shape {
    private $radius;

    public function __construct($radius) {
        $this->radius = $radius;
    }

    public function area() {
        return pi() * pow($this->radius, 2);
    }
}

class Rectangle extends Shape {
    private $width;
    private $height;

    public function __construct($width, $height) {
        $this->width = $width;
        $this->height = $height;
    }

    public function area() {
        return $this->width * $this->height;
    }
}

$shapes = [new Circle(5), new Rectangle(4, 6)];

foreach ($shapes as $shape) {
    echo $shape->area() . "\n";
}
?>


Both Circle and Rectangle implement area(), but the output depends on the object type.
